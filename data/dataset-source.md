# Dataset Source

**Dataset:** WHO Global Air Pollution Dataset  
**Publisher:** World Health Organization (WHO)

## What it contains
Global air quality measurements across monitored locations with pollutant indicators and AQI metrics.

## Key metrics used in this project
- PM2.5 concentration (mean)
- AQI values (mean)
- Other pollutant AQI values (CO, NO₂, O₃) where applicable

## Notes
- This repository does not include the full raw dataset to avoid large file uploads and to respect source usage constraints.
- Cleaned/aggregated extracts were created in Excel prior to visualization in Flourish.
